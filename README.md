# Introduction
Thuymai is a reward based permissioned blockchain solution for Waste management.

## Users
   - Waste Generator
      
   - Community Bin       
      
   - Transporter

   - Recyclers
   
   - LandFill Manager
   
   - Municipality / Corporation
